# I4.0_Project
#Industry 4.0 Cyber-Physical System for Smart Machine Monitoring.

Developed an Industry 4.0 IoT-based predictive maintenance system using ESP32, integrated with temperature, vibration, current, and IR sensors for real-time machine monitoring. Implemented a Random Forest machine learning model for anomaly detection and intelligent decision-making, along with a ThingSpeak dashboard for cloud-based visualization and automated safety actions such as alerts and motor shutdown.
